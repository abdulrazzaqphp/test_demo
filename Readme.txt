Git Basic Commands:

git --version       (for checking git version)
git help config     (opening the helping page on browser)
git config --help   (opening the helping page on browser)
mkdir test          (create directory that name is test)
cd test             (git into the test directory)
touch demo          (create a file that name is demo)
git status          (check the status of the current directory)
git add demo.txt    (add file into staging area)
git commit -m "message" (add message into git updatation)
git config --global user.username github-username   (connect github account)
git remote add origin linkOfNewRepostory (Add path of new repository)
git push origin master  (upload local directory file into github repository)
